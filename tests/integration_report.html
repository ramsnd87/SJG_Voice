Issues found and resolved

  1. EnginSam TTS not running — booted tts_server.py via the local venv. Chatterbox loaded on CUDA, model ready, sample rate 24 kHz.
  2. Ultron voice missing from EnginSam TTS — the SAM_PODCAST mirror existed but the actual TTS engine reads from EnginSam\tts-server\voices\. Added (no overwrite) ultron profile next to sam and jack_podcast. EnginSam now lists 3 voices.
  3. RAM logo broken in banner — renderer is served over http://, browsers block file:// resources. Added a sidecar route GET /api/asset/ram-logo that streams the PNG with proper image/png headers. Banner enlarged (64×64), framed, drop-shadow tuned.
  4. Vite bound to localhost (IPv6) while Electron / wait-on used 127.0.0.1 (IPv4) — Electron hung at startup. Fixed by setting server.host = "127.0.0.1" in vite.config.ts.
  5. Duplicate ultron entries in the library — voice exists in both SAM_PODCAST mirror and EnginSam engine roots; UI showed it twice. Sidecar listLocalVoices() now dedupes by slug, EnginSam-engine wins. Library lists 4 unique voices.

Earlier iteration — Sound Design FX

  • 11 new sliders per mood on top of Exaggeration / cfg_weight, grouped into Playback · EQ · Color · Echo & Stereo.
  • Playback: Rate (0.5–1.5×) · Volume (0–2×, master gain).
  • EQ: Bass (low shelf, 250 Hz, ±12 dB) · Presence (peaking, 2 kHz, ±12 dB) · Treble (high shelf, 4.5 kHz, ±12 dB).
  • Color: Drive (waveshaper saturation) · Reverb mix (convolver with synthetic IR) · Compression (one-knob: threshold + ratio).
  • Echo & Stereo: Echo mix · Echo time (40–800 ms) · Stereo pan (-1 left … +1 right).
  • Live audio chain built with the Web Audio API: MediaElementSource → drive → 3-band EQ → comp → split (dry / convolver / delay+feedback) → stereo pan → master gain → destination. Slider changes update parameters in real time on the playing audio.
  • "Reset FX to flat" button restores all 11 dials to neutral without losing your sample line or synthesis values.
  • Persistence: the full fx object is saved alongside Exaggeration / cfg_weight in the per-voice override file. Reload returns to the exact mix you set.
